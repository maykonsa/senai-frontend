CSS DISPLAY

O CSS Display é uma propriedade utilizada para indicar a forma como os elementos HTML serão dispostos na página ou em espaços delimitados. Justamente por essas características, ele se torna um recurso essencial para manter a organização dos elementos da página.

Propriedades Display

A propriedade "Display" possui inúmeras variações, porém neste primeiro momento iremos trabalhar com apenas as três mais básicas (utilizadas para controlar os tipos de elementos):

  • Block: elemento em bloco. Este ocupa 100% da largura disponível e sempre será iniciado em uma nova linha. Exemplo: Títulos (h1 ao h6), Divs, parágrafos...;
  • Inline: elemento em linha. Ocupa apenas o espaço necessário para exibir o seu conteúdo. Exemplo: Links e Imagens;
  • Inline-block: elemento de blocos em linha. Isto é, irá permitir a alteração tanto da largura quanto da altura de seus elementos (colocando-os em linha).
