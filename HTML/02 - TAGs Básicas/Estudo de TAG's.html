TAG'S de estilo (obsoletas)

Desde muito tempo utilizamos alguns recursos para dar o devido destaque a um trecho em um texto. Estes recursos são conhecidos como "ESTILOS", onde variam em: Negrito, Itálico, e Sublinhado.

O recurso de Negrito é utilizado quando queremos destacar um texto deixando-o mais espesso "grosso". A tag responsável pelo negrito é a "<b>". Importante ressaltar que esta TAG encontra-se obsoleta, porém só iremos tratar disto no assunto sobre "SEMÂNTICA".

O Itálico é um recurso que visa destacar o texto colocando-o em uma leve inclinação "<i>".

Sublinhar um texto é o ato de destacá-lo inserindo um traço sobre o texto desejado. A TAG que permite está ação é a "<u>".

TAGS de lista

As TAG'S de lista são usadas para criar listas ordenadas e não-ordenadas. Onde as TAG's responsáveis por estes recursos são <OL>para listas ordenadas, <UL> para listas não odernadas e <LI> para dada item da lista (seja ordenada ou não).

Lista ordenadas, usadas quando a ordem é relevante:

  1. AAAA
  2. BBbbbbbBBBBBBBBB
  3. CCCCCCCCCC
  4. DdDdDdDdDdd
  5. EEEEEEEEeeeeeeeeeeee

Lista não ordenadas, usadas quando a ordem é irrelevante:

  • aaaaa
  • BBBBBBBB
  • cccccc
  • DDDDDDDDDD
  • EeEeEEEEeeeEeee
