Imagens

Sendo uma imagem parte de um processo de comunicação, podemos dizer seguramento que uma imagem é uma representação, ou seja, é uma forma de comunicar a outrem, através de uma mediação, aquilo que nós vemos (ou receber de outro aquilo que eles vêem), ou seja, pensam, conscebem e/ou imaginam.

Em se tratando de desenvolvimento de páginas HTML, para inserir uma imagem utilizamos o elemento <img>. Este elemento não possui conteúdo, ou seja, não tem sua respectiva TAG de fechamento, e por padrão trabalha com pelo menos um atributo obrigatório "src" e por padrão também "alt".
